# Photography Course

**14-Day Challenge: Improve Skills + Discover Your Style**

**Gear:** Sony ZV-E10 II + FE 24-50mm f/2.8 G  
**Location:** Charleston, SC / Isle of Palms  
**Daily:** 30-60 min blue-hour shoot → Lightroom edit → 1 image to Instagram @richardsawyers + full set here  
**Start:** July 23, 2026

## How It Works
1. Read the daily lesson (link below)
2. Shoot during morning blue hour
3. Edit in Lightroom (Mac or iPad)
4. Upload best photo(s) + reflection.md to the Day-XX folder
5. Commit & push
6. Paste the Day folder link in our chat for AI review

## Style Discovery
Every day note:
- What felt natural?
- What felt forced?
- Preferred color/mood/subject?

Mid-course (Day 7) and final (Day 14) style audit.
