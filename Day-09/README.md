# Day 09 – Kids & Connection

**Genre focus:** Portrait
**Time:** Flexible (blue hour preferred if kids cooperate)
**Location suggestions:** Park, beach, backyard.

## Study Material
- Get down to their eye level.
- Capture genuine interaction rather than forced smiles.

## Shooting Brief
1. Continuous AF, higher shutter if they move.
2. Shoot moments between the “poses”.
3. Include context when it adds to the story.

## Upload
Best 3–5 + reflection.md
